
Xbox One and Xbox 360 controller supported: Connect via bluetooth or wire before running game.

Keyboard Controls:

WASD, arrow keys for movement.
Spacebar to jump.
P to pause.
Press Esc to quit.
Use mouse to click onscreen buttons.

Game works best on 16:9 screens.
Designed for 2560 x 1440, should work at any resolution with right ratio (except pause quit button and healthbar size/position.)

If daisy chaining multiple monitors, primary monitor resolution will be used, even if focus is on another monitor, so keep primary monitor in focus when running game.
(You can alt-tab out.)
